# 自动账本 Roadmap

## P0 Harness 与安全

- 本地化 Supabase 配置，避免 key 上传 GitHub。
- 建立多 Agent 角色、脚本和质量门禁。
- 每次提交前运行密钥扫描、单元测试和 debug 构建。

## P1 日常可用性

- 优化手动记账表单：分类选择、账户选择、备注、收入/退款/转账类型。
- 待确认箱批量确认/忽略。
- 捕获通知后去重，避免同一通知重复入账。
- 增加常见银行、微信、支付宝、云闪付、外卖、电商样例解析测试。

## P2 数据完整性

- Room schema 版本化和迁移测试。
- Supabase 增量拉取，避免每次全量读取。
- 本地备份恢复：CSV/JSON 导入。
- 换机密钥导出和二维码展示。

## P3 统计体验

- 月度预算、分类预算、超支提醒。
- 日历视图、商户排行榜、账户趋势。
- 自定义分类管理。
- 首页卡片可配置。

## P4 发布质量

- Release 签名配置本地化。
- APK 安装烟测脚本。
- 崩溃日志采集和用户可读错误提示。
